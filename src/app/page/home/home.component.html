Angular application (latest version) that consumes the next API Rest "https://rickandmortyapi.com/" and does the following:
  • Implement Redux and save the data and actions of the events that are taking place in the store.
  • Loading pages by lazy loading.
  • Character page:
    1. Show character list
    2. when clicking on a character redirects to the character's detail.
  • Locations screen:
    1. Show list of locations
    2. when clicking on a location, show the list of characters.
  • Episode screen:
    1. Show episode list
    2. when clicking on an episode, show the list of characters.
  • Search: Implement a search that, depending on the page you are on, filters by "name" characters, locations of episodes.
  • Nice to have: perform unit tests with 20% of coverage
